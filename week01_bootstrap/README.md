# week01_bootstrap

## Files

- `train.py`: runnable training script
- `experiment.md`: loss / learning rate notes
- `notes.md`: environment, changes, open issues

## Run

```bash
python week01_bootstrap/train.py
```

## Change

Start with:

- `lr`
- parameter initialization
- number of steps

Keep changes in:

```text
week01_bootstrap/
```

## Record

Update:

- `experiment.md`
- `notes.md`

Track:

- command
- changed value
- loss pattern
- error or blocker
